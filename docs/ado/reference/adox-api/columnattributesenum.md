# <a name="columnattributesenum"></a>ColumnAttributesEnum
[列](./column-object-adox.md)の特性を指定します。  
  
|定数|値|説明|  
|--------------|-----------|-----------------|  
|**adColFixed**|1|列は固定長です。|  
|**adColNullable**|2|列に null 値が含まれている可能性があります。|
