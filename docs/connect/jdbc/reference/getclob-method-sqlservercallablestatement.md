# <a name="getclob-method-sqlservercallablestatement"></a>getClob メソッド (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  指定された JDBC BLOB パラメーターの値を Java プログラミング言語の Clob オブジェクトとして取得します。  
  
## <a name="overload-list"></a>オーバーロードの一覧  
  
|名前|説明|  
|----------|-----------------|  
|[getClob (int)](../../../connect/jdbc/reference/getclob-method-int.md)|パラメーターに渡されたインデックスを使用して、指定された JDBC BLOB パラメーターの値を Java プログラミング言語の Clob オブジェクトとして取得します。|  
|[getClob (java.lang.String)](../../../connect/jdbc/reference/getclob-method-java-lang-string.md)|パラメーターに渡された名前を使用して、指定された JDBC BLOB パラメーターの値を Java プログラミング言語の Clob オブジェクトとして取得します。|
