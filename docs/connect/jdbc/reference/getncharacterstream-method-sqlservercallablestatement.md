# <a name="getncharacterstream-method-sqlservercallablestatement"></a>getNCharacterStream メソッド (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

|----------|-----------------|
