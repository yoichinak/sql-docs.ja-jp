# <a name="mdx-data-definition---drop-subcube"></a>MDX データ操作 - DROP SUBCUBE


  指定したサブキューブを削除し、指定した名前を持つ以前に定義したキューブまたはサブキューブ定義に戻します。  
  
## <a name="syntax"></a>構文  
  
```  
  
DROP SUBCUBE Subcube_Name   
```  
  
## <a name="arguments"></a>引数  
 *Subcube_Name*  
 サブキューブ名を提供する有効な文字列式です。
