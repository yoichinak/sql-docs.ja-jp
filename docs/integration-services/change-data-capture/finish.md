# <a name="finish"></a>[完了] 

[!INCLUDE[sqlserver-ssis](../../includes/applies-to-version/sqlserver-ssis.md)]


  **[完了]** ページには、ウィザードの処理がまとめて表示されます。 **[完了]** をクリックしてウィザードを閉じると、CDC インスタンスの作成ウィザードで定義したすべてのテーブルが生成されます。
