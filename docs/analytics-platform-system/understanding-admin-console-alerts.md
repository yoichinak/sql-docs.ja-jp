# <a name="understanding-admin-console-alerts-in-analytics-platform-system"></a>Analytics Platform System の管理コンソールのアラートについて
アラートはアプライアンス **管理コンソール** と **System Center Operations Manager** (SCOM) に表示されます。 このアラートの一覧を使用して、追加の調査が必要なアラートを特定できます。  
  
Internet Explorer を使用して **管理コンソール** に接続する方法については、「 [管理コンソールを使用したアプライアンスの監視 &#40;Analytics Platform System&#41;](monitor-the-appliance-by-using-the-admin-console.md)」を参照してください。 **SCOM**の詳細については、「 [System Center Operations Manager &#40;Analytics Platform System を使用したアプライアンスの監視](monitor-the-appliance-by-using-system-center-operations-manager.md)」を参照してください&#41;  
  
Transact SQL を使用したアラート情報の取得の詳細については \- 、「 [システムビューを使用したアプライアンスの監視 &#40;Analytics Platform System&#41;](monitor-the-appliance-by-using-system-views.md)」を参照してください。  
  
## <a name="types-of-alerts"></a>アラートの種類  
通常の状態を示すアラート名は、通常、調査を必要としません。 アラート名には、NON_CRITICAL が必要になることがあります。 他のすべての種類のアラートには調査が必要です。
