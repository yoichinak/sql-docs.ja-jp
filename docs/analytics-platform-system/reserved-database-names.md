# <a name="reserved-database-names-in-parallel-data-warehouse"></a>Parallel Data Warehouse での予約済みデータベース名
予約されたデータベース名は、SQL Server PDW のユーザーデータベースに割り当てることができない名前です。  
  
予約されているデータベース名の一覧:  
  
-   master  
  
-   model  
  
-   msdb  
  
-   tempdb  
  
-   pdwtempdb  
  
-   pdwtempdb1  
  
-   pdwtempdb2  
  
-   DWConfiguration  
  
-   DWDiagnostics  
  
-   DWQueue
