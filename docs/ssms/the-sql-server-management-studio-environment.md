# <a name="the-sql-server-management-studio-environment"></a>SQL Server Management Studio 環境
[!INCLUDE[SQL Server Azure SQL Database Synapse Analytics PDW ](../includes/applies-to-version/sql-asdb-asdbmi-asa-pdw.md)]
[!INCLUDE[ssManStudioFull](../includes/ssmanstudiofull-md.md)] は、 [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)]のクエリの管理と開発のための豊富な機能を提供する環境です。  

> [!NOTE]
> グラフィックス ドライバーとの競合が SQL Server Management Studio レンダリング プロセスに問題を引き起こすことが見つかりました。そのため、ハードウェア グラフィックス アクセラレーションは既定で無効になっています。  SSMS インターフェイスでレンダリングや映像に関するその他の問題が発生した場合、[ツール] の [オプション] で [可能ならハードウェアのグラフィックス アクセラレータを使用する] が "*選択されていない*" ことを確認してください。
