# <a name="group-by-expression-list"></a>GROUP BY 式リスト
GROUP BY では、列名だけでなく式リストもサポートされています。 GROUP BY 句を含む SELECT ステートメントの選択リストには、GROUP BY 句または set 関数の式のみを含めることができます。
