# <a name="date-arithmetic"></a>日付の算術演算子
ドライバーは、日付、時刻、またはタイムスタンプ列からの整数の加算と減算をサポートしています。 日付列の場合は、加算または減算する日数を整数で指定します。 TIME または TIMESTAMP 列の場合、整数は加算または減算する秒数を指定します。
