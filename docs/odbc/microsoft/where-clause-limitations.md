# <a name="where-clause-limitations"></a>WHERE 句の制限事項
WHERE 句の句の最大数は40です。  
  
 LONGVARBINARY 列と LONGVARBINARY 列の長さは、最大255文字のリテラルと比較できますが、パラメーターを使用して比較することはできません。
